The Computer Science classes I've taken

  • CSCI 2033: Elementary Computational Linear Algebra

    • This is the other CSCI class I took in the Spring 2023 Semester, the main topics for this class were Linear and Geometric Algebra, along with Linear Transformations, some of the topics covered were: vectors, spaces/subspaces, linear combinations, linear independence/dependence, matrices, inner and outer product, orthogonality/orthonormal basis, and geometric product. This class was very proof-heavy, but it helped me to gain a more fundamental understanding of linear and geometric algebra.

  • CSCI 2021: Machine Architecture and Organization

    • This is one of two CSCI classes I took in the Spring 2023 Semester, the main topics were learning how to program in C, then into x86-64 Assembly code, computing Architecture, and processing systems/environment. I would say the last third of the class were especially built off of the first main topics of the course, as we studied how CPUs and Memory works, along with performance optimizations using previous knowledge of earlier topics such as pointers, registers, etc. Furthermore, we studied areas such as Virtual Memory, the ELF file format, and linking.

  • CSCI 2011: Discrete Structures

    • I also took this class in the Fall 2022 Semester, the main topics were Logic, Sets, Cryptography, Induction, Combinatorics, Relations, Graphs, and Trees. This course helped to develop my thinking and problem-solving skills, especially with proofs, that will be essential in computer science later on.

  • CSCI 1933: Intro to Algorithms and Data Structures

    • Took this class in the Fall 2022 Semester, this was taught in Java where I learned about OOP and basic data structures such as Linked Lists, Stacks and Queues, Binary Trees, and HashTables. I also learned about basic algorithms such as Bubble Sort, Binary Search, Merge Sort, Breadth-first-search and Depth-first-search. In addition, I also learned other concepts too, such as abstraction, inheritance, exceptions, and Big-O notation.

  • CSCI 1133: Intro to Computing and Programming Concepts

    • I took this class in the Spring 2022 Semester, this was taught in Python where I learned basic programming concepts such as loops, recursion, conditionals, File I/O, iterating through a string/list, etc.
